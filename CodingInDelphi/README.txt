Example code for the Coding in Delphi book by Nick Hodges: 
https://leanpub.com/codingindelphi 
https://plus.google.com/communities/110978417023349293804 

The examples are usually fully compiling Delphi projects either based on code snippets in the various book chapters, or complete new example projects. 

At the root level, the directories are named after the book chapters. 
Within those, they are organized by example.
